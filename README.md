Reto de Paradigma Digital: Microservicios de Flask para Juego de Tronos.

El reto consta de 3 microservicios: 

    Lugares
    Personajes
    Un microservicio conectando los 2 anteriores


Antes de inicializar los microservicios, hay que correr el archivo "create_databases.py
Se generaran 2 archivos: 
    places.db para los lugares
    people.db para los personajes.
    
Versión 1.0: 
Los microservicios Lugares y Personajes están en el mismo microservicio.
